🛒 MKMart – Django E-commerce Website

MKMart is a full-featured e-commerce web application built using the Django framework. It allows users to browse products, manage their cart, and securely place orders, while providing an admin panel for managing inventory and users.

🚀 Features

👤 User Features
User registration and authentication (login/logout)
Browse products by category
Product detail pages
Add to cart / remove from cart
Checkout and order placement
Order history and tracking

🛠️ Admin Features

Add, update, and delete products
Manage categories
View and manage orders
User management via Django admin panel

🧰 Tech Stack

Backend: Django (Python)
Frontend: HTML, CSS
Database: PostgreSQL 
Authentication: Django built-in auth system
